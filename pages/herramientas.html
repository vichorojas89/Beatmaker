Herramientas para trabajar tus beats


Se pueden utilizar distintos tipos de elementos para hacer tus beats, pueden ser hechos en un computador con un software DAW o en samplers que son hardware

FL Studio, anteriormente conocido como FruityLoops, es un popular software de producción musical desarrollado por la empresa belga Image-Line. Se utiliza principalmente para la creación, edición, mezcla y masterización de música electrónica, aunque también es ampliamente utilizado en otros géneros musicales como el rap, el hip-hop, el pop y más.


El software FL Studio se puede complementar con controladores midi como el que mensionaremos a continuacion:



El M-Audio Oxygen Pro 25 es un controlador MIDI compacto y versátil diseñado para músicos y productores que desean una interfaz intuitiva para controlar su software de producción musical. Aquí tienes una breve descripción de sus características principales:

  1. Teclado sensible a la velocidad y a la presión: El Oxygen Pro 25 cuenta con un teclado de 25 teclas sensibles a la velocidad y la presión, lo que permite una expresividad musical más dinámica. Esto significa que la intensidad con la que se tocan las teclas afecta el volumen y otros parámetros de sonido.
  2. Pads de percusión retroiluminados: Incorpora ocho pads de percusión retroiluminados sensibles a la velocidad que pueden usarse para disparar muestras de batería, lanzar clips o controlar otras funciones de reproducción en el software de producción musical.
  3. Controladores asignables: El Oxygen Pro 25 viene con ocho perillas asignables, nueve faders y botones de transporte, lo que proporciona un control práctico y personalizable sobre los parámetros de tu software de producción musical.
  4. Pantalla LCD integrada: Cuenta con una pantalla LCD integrada que muestra información útil como nombres de pistas, valores de parámetros y configuraciones en tiempo real, lo que facilita el control y la navegación sin necesidad de mirar la pantalla de tu computadora.
  5. Botones de control de DAW: Posee botones dedicados para controlar funciones específicas de tu software de producción musical, como reproducción, grabación, bucle, transporte y navegación entre pistas, lo que mejora la eficiencia y el flujo de trabajo.
  6. Conectividad versátil: El Oxygen Pro 25 se conecta a tu computadora a través de USB y es compatible con una amplia gama de software de producción musical, incluidos los principales DAW como Ableton Live, FL Studio, Pro Tools y más.
